# ⚡ CV RAPIDO — Generador Profesional de CV

**CV RAPIDO** es la solución definitiva para crear currículums profesionales de forma gratuita, ultraliviana y en menos de 3 minutos. Diseñado específicamente para responder a las demandas del mercado laboral actual en Argentina.

## 🎯 ¿Qué es CV RAPIDO?
Es una plataforma web minimalista que permite a cualquier buscador de empleo generar un CV de alta calidad sin complicaciones. A diferencia de otras herramientas, CV RAPIDO se enfoca en la **velocidad** y la **profesionalidad**, eliminando barrearas como el registro obligatorio o formularios infinitos.

## ✨ Características Destacadas
- **Sin Registro:** Tu tiempo es valioso. Entrá, completá y descargá.
- **Categorías Laborales Específicas:** Optimizado para rubros como Minería, Gastronomía, Construcción, Tecnología y más.
- **Diseño Empresarial:** 10 plantillas profesionales diseñadas para ser legibles, limpias y aceptadas por reclutadores.
- **Privacidad Primero:** Almacenamiento temporal de datos (TTL 24h) para garantizar que tu información no quede expuesta.
- **Optimizado para el Norte Argentino:** Adaptado a los perfiles laborales más buscados en la región.

## 🛠️ Tecnología de Vanguardia
Construido con un enfoque en el rendimiento:
- **Core:** JavaScript Vanilla (sin frameworks pesados para una carga instantánea).
- **Diseño:** CSS Modular y dinámico.
- **Backend:** Node.js con generación directa de PDF.
